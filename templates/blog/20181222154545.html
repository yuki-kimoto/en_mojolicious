Change Mojolicious's home directory

Use the "MOJO_HOME" environment variable to change the Mojolicious home directory. 
BEGIN {
   $ENV {MOJO_HOME} ='/ foo / bar';
}
use Mojolicious::Lite;
Let's set it before loading Mojolicious::Lite. Alternatively, set environment variables from the shell in advance. 
export MOJO_HOME = / foo / bar
