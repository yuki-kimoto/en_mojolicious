Mojolicious plugin for viewing SQLite information Mojolicious :: Plugin :: SQLiteViewerLite
Notice Mojolicious :: Plugin :: SQLiteViewerLite has been integrated into Mojolicious :: Plugin :: DBViewer . Mojolicious :: Plugin :: SQLiteViewerLite has been released on CPAN. With the Mojolicious plugin, you can view the SQLite table list and table definitions in your browser. It may be helpful to think of it as a Perl version of SQLite dedicated to viewing phpMyAdmin. The function is much simpler. This is the SQLite version of the useful Mojolicious :: Plugin :: MySQLViewerLite that was released the other day. It can be used with the same interface.
  • Mojolicious :: Plugin :: SQLiteViewerLite >

    How to use 
    #Mojolicious :: Lite
plugin'SQLiteViewerLite', dbh => $ dbh;

#Mojolicious
$ app-> plugin ('SQLiteViewerLite', dbh => $ dbh);

#Access
http: // localhost: 3000 / SQLiteviewerlite
